报备客户接口

报备客户接口

报备客户接口

接口调用请求说明

http请求方式: POST

http://api.ehaofang.com/ehaofang-pubapi/xtCustomer/customerperReserve.do

参数说明

参数名 是否必须 类型 说明
memberId 是 String 接入方公司的经纪人ID
memberName 是 String 经纪人姓名
mobilephone 是 String 经纪人联系电话
sex 否 String 经纪人性别 //（0-未知，1-男，2-女）
email 否 String 经纪人邮箱
memberAddress 否 String 经纪人地址
name 是 String 报备客户名称
firstNum 否 String 手机号前三位 phone参数为空时必须字段
lastNum 否 String 手机号后四位 phone参数为空时必须字段
propertyId 是 String 楼盘ID
lookHomeDate 是 String 来访时间 //格式如：2016-12-10 07:00
visitorsNumber 是 String 来访人数
saleUserId 否 String 案场ID
shuttle 是 String 来访方式 0班车 1自驾
agentId 是 String 接入公司的分销公司ID
branchId 是 String 接入公司的分销公司门店ID
address 否 String 上车地点 （班车时为必须参数）
carTime 否 String 上车时间 （班车时为必须参数）//格式如：07:00
decoration 是 String 描述
phone 否 String firstNum lastNum参数为空时 为必须字段
accessToken 是 String 接口凭证
